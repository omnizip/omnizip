== 7z Format

=== General

The 7z format is a compressed archive format created by 7-Zip.

=== Features

* High compression ratio (LZMA2)
* Solid archives
* Multi-volume support
* AES-256 encryption

=== Usage

[source,bash]
----
# Create 7z archive
omnizip archive create backup.7z documents/

# Extract 7z archive
omnizip archive extract backup.7z output/
----
